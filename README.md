# Porch Updates

Public metadata and release download hosting for Porch.

GitHub Pages serves update feeds from `updates.porch.chat`. Large desktop
installers are published as GitHub Release assets because the Windows installer
is larger than GitHub's normal per-file git limit.

Stable download URLs:

- `https://github.com/porch-chat/porch-updates/releases/latest/download/PorchSetup.exe`
- `https://github.com/porch-chat/porch-updates/releases/latest/download/Porch.AppImage`
- `https://github.com/porch-chat/porch-updates/releases/latest/download/porch-linux-amd64.deb`
